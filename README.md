# WI Bulk Mover
A simple extension that allows you to move multiple WI entries from one lorebook into another.

## Installation
- ST `+=+1.12.4` required
Install the extension using this link: `https://github.com/leandrojofre/SillyTavern-WI-Bulk-Mover.git`

## Features
- Clone **all/some** entries from `lorebook_a` into `lorebook_b`
- Transfer **all/some** entries from `lorebook_a` into `lorebook_b`
- Delete **all/some** entries from `lorebook_a` It has a **confirm deletion** screen
